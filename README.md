# Buried Acorns

This is the home of the squirrel collectives' Acorn trove.

## Project Structure

- `Acorn_Genesis_Beta/`: Core module for the project
  - `ux/`: User experience components
  - `console/`: Console interface
    - `ux/`: Console-specific UX
    - `acorn_console.py`: Main console script

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/whitecatlord/buried-acorns-.git
cd buried-acorns-
pip install -r requirements.txt
```

## Usage

Run the console:

```bash
python Acorn_Genesis_Beta/console/acorn_console.py
```
